# Rachana's Portfolio

A personal portfolio website showcasing my experience, projects, education, and technical skills.

## Overview

Single-page portfolio built to highlight my background as a software engineer — including professional experience, academic achievements, leadership roles, and the technologies I work with.

## Tech Stack

- **React 18** — component-based UI
- **Vite 5** — fast dev server and build tool
- **Tailwind CSS 3** — utility-first styling
- **Google Fonts** — Inter (sans-serif) + JetBrains Mono (monospace)
